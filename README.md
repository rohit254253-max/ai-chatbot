# 🤖 AI Chatbot using Flask

This is a simple AI Chatbot project built using Python Flask, HTML, CSS, and API integration.

## 🚀 Features

- User-friendly chatbot UI
- Flask backend
- API integration
- Dynamic bot responses
- Responsive design

## 🛠️ Tech Stack

- Python
- Flask
- HTML
- CSS
- Requests Library

## ▶️ How to Run

1. Clone repository

```bash
git clone https://github.com/rohit254253-max/ai-chatbot.git

2.Create Virtual Environment
```bash
py -m venv venv

3.Activate Virtual Environment
```bash
.\venv\Scripts\Activate.ps1

4.Install dependencies
```bash
pip install flask requests

5.Run project
```bash
python app.py

6.Open Browser
http://127.0.0.1:5000
